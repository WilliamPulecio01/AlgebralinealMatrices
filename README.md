# 📐 Trabajo Final — Álgebra Lineal
Verificación de temas de Álgebra Lineal usando Python con las librerías **NumPy** y **SymPy**.

---

## 📁 Contenido del repositorio

| Archivo | Tema | Librería |
|--------|------|----------|
| `T1_operaciones_matrices.py` | Suma, resta, escalar y multiplicación de matrices (≥ 3×3) | NumPy |
| `T2_sistemas_ecuaciones.py` | Sistemas con solución única, infinitas soluciones e inconsistentes | NumPy |
| `T3_matrices_especiales.py` | Matrices rotacional, simétrica y antisimétrica | NumPy |
| `T4_determinantes.py` | Determinantes por Gauss, diagonalización y cofactores | NumPy |
| `T5_propiedades_determinantes.py` | Las 9 propiedades de los determinantes | NumPy |

---

## ⚙️ Requisitos

Tener instalado **Python 3** y las siguientes librerías:

```
numpy
sympy
```

---

## 🚀 Instalación y uso

**1. Clonar el repositorio**
```bash
git clone https://github.com/tu-usuario/algebra-lineal.git
cd algebra-lineal
```

**2. Instalar las dependencias**
```bash
pip install -r requirements.txt
```

**3. Correr cualquier archivo**
```bash
python T1_operaciones_matrices.py
python T2_sistemas_ecuaciones.py
python T3_matrices_especiales.py
python T4_determinantes.py
python T5_propiedades_determinantes.py
```

---

## 📚 Temas cubiertos

- ✅ Operaciones con matrices (suma, resta, escalar, multiplicación)
- ✅ Multiplicación verificada por: Sistemas de Ecuaciones, Cofactores y Gauss-Jordan
- ✅ Sistemas de ecuaciones lineales (solución única, infinitas, inconsistente)
- ✅ Matrices rotacionales, simétricas y antisimétricas
- ✅ Determinantes por tres métodos distintos
- ✅ Las 9 propiedades de los determinantes con 2 ejemplos cada una
